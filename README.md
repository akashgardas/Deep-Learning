# Deep Learning Study Repository

A comprehensive repository documenting my deep learning journey, featuring implementations, projects, and notes from structured coursework. This repo serves as both a learning resource and a portfolio of hands-on deep learning work.

## 📚 Overview

This repository contains code, models, and documentation covering core deep learning concepts and their practical applications. The work progresses from foundational neural network theory through modern architectures, with emphasis on computer vision tasks using TensorFlow/Keras.

## 🎯 Topics Covered

- **Neural Networks Fundamentals**: Perceptrons, multi-layer perceptrons (MLPs), and tensor operations
- **Training Mechanics**: Activation functions, optimizers, loss functions, and backpropagation
- **Regularization Techniques**: Dropout, model regularization, and preventing overfitting
- **Convolutional Neural Networks (CNNs)**: Architecture, image classification, and custom datasets
- **Computer Vision Applications**: Image classification on CIFAR-10, MNIST, and custom datasets

## 🛠️ Technology Stack

- **Framework**: TensorFlow / Keras
- **Languages**: Python
- **Libraries**: NumPy, Matplotlib, scikit-learn
- **Tools**: Jupyter Notebooks, Git

## 💡 Key Projects

### Image Classification with Custom Data

A mini-project implementing a CNN classifier on custom datasets, demonstrating the complete pipeline from data preparation to model evaluation.

### CIFAR-10 & MNIST Classification

Standard benchmarks for evaluating CNN performance on well-known image datasets.
